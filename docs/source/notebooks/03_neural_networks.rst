Neural Networks Notebooks
==========================

Notebooks 08–10 build neural network architectures from scratch using Flax.

.. toctree::
   :maxdepth: 1

Notebook 08 – MLP from Scratch
--------------------------------

**File**: ``notebooks/03_neural_networks/08_mlp_from_scratch.ipynb``

Topics covered:

* Defining a Flax ``nn.Module``.
* Weight initialisation strategies (Glorot, He).
* Activation functions: ReLU, GELU, SiLU, tanh.
* Forward pass and parameter inspection.

Notebook 09 – Minimal CNN
---------------------------

**File**: ``notebooks/03_neural_networks/09_cnn_minimal.ipynb``

Topics covered:

* 2-D convolutions with ``nn.Conv``.
* Max pooling and average pooling.
* Feature map visualisation.
* Building a simple image classifier.

Notebook 10 – Attention from Scratch
--------------------------------------

**File**: ``notebooks/03_neural_networks/10_attention_from_scratch.ipynb``

Topics covered:

* Scaled dot-product attention.
* Multi-head attention mechanism.
* Positional encoding.
* Full Transformer encoder layer (attention + FFN + LayerNorm).
